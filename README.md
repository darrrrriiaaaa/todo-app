# Todo app

This is a full-stack web app that allows users to add tasks to their todo-list. Built with React, Express, and MongoDB.

## Features

- create, edit, and delete tasks
- change the status (done/undone)
- search for a specific task by its name or description
- filter tasks by status
- sort tasks by their priority

## Technologies

**Frontend:** React, Tailwind
**Backend**: Node.js + Express, MongoDB, cors, dotenv

## Installation

1. Clone the repository.
```
git clone https://github.com/darrrrriiaaaa/todo-app.git
cd todo-app
```
2. Install dependencies for backend and start the server.
```
cd backend
npm install
node server
```
3. Open another terminal with todo-app folder. Install dependencies for frontend and run the client.
```
cd frontend
npm install
npm run dev
```
4. Open the link in the console.

## Usage

Fill in the form to add a new task and click on the button to add it. Click on the settings button on any task, and change any options you want. Click on the delete icon to delete the task from the board. Click on the checkbox to change the status of the task. Use a search input at the top of the page. Change the filters for status and sorting by priority. Click on Reset button to clear all filters.
